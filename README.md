# Codemaster

## Descrição
A estrutura do site já estava completa em HTML e CSS, porém algumas funcionalidades interativas não estavam funcionando corretamente.  
O objetivo deste trabalho foi corrigir e implementar essas funções utilizando **JavaScript**, garantindo que os elementos do site funcionem de forma adequada e melhorem a experiência do usuário.

## Funcionalidades Corrigidas

### 1. Menu Mobile[X]
Foi implementado o funcionamento do menu para dispositivos móveis.  
Ao clicar no ícone do menu, a navegação abre e fecha corretamente.

### 2. Navegação Ativa[X]
A navegação agora destaca automaticamente o link da seção que o usuário está visualizando na página.

### 3. Modo Escuro / Claro[X]
Foi adicionada a funcionalidade de alternar entre **modo claro** e **modo escuro**, permitindo ao usuário escolher o tema da interface.

### 4. Animação do Título[X]
O título principal da página recebeu uma animação para melhorar o aspecto visual ao carregar o site.

### 5. Animação da Seção Home[X]
A seção inicial (Home) agora possui animações que aparecem quando a página é carregada.

### 6. Animação das Seções[X]
As demais seções do site possuem animações que são ativadas conforme o usuário rola a página.

### 7. Botão de Voltar ao Topo[X]
Foi implementado um botão que aparece durante a rolagem da página e permite retornar rapidamente ao topo do site.

### 8. Carrossel[X]
O carrossel foi corrigido para permitir a navegação entre os elementos de forma automática ou manual.

## Tecnologias Utilizadas
- HTML
- CSS
- JavaScript
- Manipulação do DOM

## Objetivo
Garantir que todas as funcionalidades interativas do site funcionem corretamente utilizando **JavaScript**, além de melhorar a usabilidade e a experiência do usuário.
